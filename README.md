# arcpy-global-atlas-generator (arcpy)
## Opis projektu
Projekt realizuje automatyzację procesu generowania wielostronicowych atlasów map.

## Wykorzystane technologie i narzędzia
* ** Oprogramowanie:** ArcGIS Pro 3
* ** Język programowania: ** Python 3
* **Biblioteki GIS:** 'arcpy' (moduły 'arcpy.md' oraz 'arcpy.da')
* **Dane źródłowe:** Natural Earth Data (Admin 0 - Countries)
